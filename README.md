# AI Social Media Content Creator 📱

## Project Overview

AI Social Media Content Creator is an AI-powered web application that automatically generates social media marketing content for businesses.  
The user enters a business type, and the application creates Instagram captions, LinkedIn posts, Facebook posts, hashtags, and a 7-day content plan using Gemini AI.

---

## Features

- Generate Instagram captions
- Generate LinkedIn marketing posts
- Generate Facebook posts
- Generate relevant hashtags
- Create 7-day social media content ideas
- Simple and interactive Streamlit interface
- AI-powered content generation using Gemini API

---

## Technologies Used

- Python
- Streamlit
- Google Gemini API
- Python-dotenv
